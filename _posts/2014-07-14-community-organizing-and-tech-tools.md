<p>I was recently talking with a friend about organizing an event to spur up some community involment in local politics. I promised my friend I would look for some tools that would make online organizing the event and people easier. </p>

<p>To my despair I found a blackhole of despair because all the software out there was either incredibly difficult to use (mainly because of design) or outrageously priced. After an afternoons worth of research I simply gave up because I was wasting more time researching this problem that it was taking away from the actual organizing. </p>

<p>I still began to think about the problem of <em>Community Organizing and tech tools</em> mainly how it seems like there should be tech tools that fulfill the needs of people who are doing community organizing. But a quick Google search will prove how this is not the case. I'm not completely sure why the current market for software in this space is so bland but it could use some innovation.</p>
